# Weather API App

## Purpose
To provide the user with a quick way to get the current weather as well as an upcoming 5 day forecast.  The app also saves the last 6 searches.

## Built With
* HTML 
* Bootstrap and CSS
* Javascript

## API's Used
* OpenWeather One Call API
* OpenWeather Current Weather API
